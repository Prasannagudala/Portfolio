Prasanna In a Nutshell

My skills include performing data analysis and visualization in Python. I can derive and communicate insights from complex datasets using analytic tools and programming languages. I also work on making predictive models using supervised machine learning techniques, and have a foundation in mathematics and statistics.

Currently, I serve as a Program Associate at WASSAN (Watershed Support Services and Activities Network), where my role centres around providing data insights and leveraging advanced technology, specifically vision-based deep learning, for agricultural applications. My key projects include Plantation Casualty Monitoring, Crop Identification, Crop Area Estimation, and comprehensive data analysis and visualization.A person who always strives for excellence, I go to every length to ensure that every project I work on is seen to the end.

I am deeply committed to delivering excellence in every project I undertake, ensuring thoroughness and attention to detail from inception to execution. My approach is marked by a relentless pursuit of innovation, constantly seeking opportunities to enhance processes and outcomes through data-driven strategies.

My academic journey includes a master’s degree in Data Science from Christ University, where I acquired a robust theoretical foundation complemented by hands-on experience in real-world data analytics challenges. This educational background, coupled with my practical expertise, equips me with a comprehensive skill set to tackle the complexities of data analysis in any setting.

Projects & Publications

Next Word Prediction

In this project, I amalgamated data from various sources to develop a predictive model for anticipating the next word in a sequence. Employing Natural Language Processing (NLP), the focus was on accurately forecasting the subsequent word based on user-provided preceding words. Rigorous data preprocessing and model optimization culminated in a versatile tool capable of enhancing text completion and powering interactive chatbots with contextually appropriate word predictions.

Skills demonstrated: Python, NLP, LSTM, RNN.

Loan Approval Prediction: Naive Bayes

In this project, I developed a Loan Prediction System using Python. This system allowed applicants to apply for loans and receive notifications when their loans were approved. The approval process was automated, using data provided by the applicant to determine the likelihood of loan approval. The goal was to streamline the loan application process and provide quick feedback to applicants.

Skills demonstrated: Python, ML algorithms, EDA

HR Analytics: Data Visualization

In this project, I aimed to gain insights into employee attrition within the tech sector by visualizing HR analytics data using Power BI. The primary goal was to identify key factors contributing to employee attrition and to enable HR departments to make informed decisions that can improve employee retention and satisfaction.

Skills demonstrated: Data cleaning, Power BI, Data visualization

Plantation Casualty Monitoring: Deep Learning

In this project, a plantation casualty monitoring system was developed. Data collection was performed using a DJI Mini 2 drone, and object detection was employed to distinguish between healthy and dead plants. This methodology assisted in agricultural management by identifying areas requiring attention.

Skills demonstrated: Data collection, Computer vision, Deep learning, Object detection

Sales Analysis

In this project, the data was collected from Kaggle and analyzed to identify higher sales purchase rates.

Skills demonstrated: Data collection, EDA, Python, Data cleaning, Excel, Visualization, Charts.

Chatbot

In this project, I collected and curated data in accordance with user specifications, employing natural language processing (NLP) techniques to develop a chatbot capable of autonomously generating responses tailored to user queries.

Skills demonstrated: Python, NLP, Mistral Model.

Deep Learning - Crop Identification

In this project, data was gathered using the DJI Mini 2 drone to capture detailed aerial imagery of agricultural fields. I successfully performed crop identification, utilizing the high-quality data to accurately distinguish different crop types.

Skills demonstrated: Python, Deep learning, CD-CNN Model, Data Collection

Identifying weeds in crops

In this project, I aimed to detect weeds in crops from agricultural fields. Images of weeds and crops were collected from Kaggle and processed using image processing techniques with YOLOv7 algorithms. These algorithms enabled accurate weed detection, making it easier for farmers to identify and manage weed crops effectively.

Skills demonstrated: Python, Data collection, Deep Learning, Computer Vision

Publication: Springer
